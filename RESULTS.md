# Results Summary — FinNutriAgent

This document summarizes the key quantitative results from the experiments described in the associated paper. All results are fully reproducible using `example/demo_finagent.ipynb`.

---

## 1. Optimization Feasibility

| Metric | Value |
|---|---|
| Households tested | 100 |
| Solutions with status `Optimal` | 100 (100%) |
| Solver | PuLP / CBC (deterministic) |
| Typical solve time | < 1 second |
| Food items in candidate pool (halal) | 310 / 350 |

All 100 households achieved feasible, optimal solutions under their respective weekly food budgets and nutritional constraints.

---

## 2. Budget Utilization

| Statistic | Value (SAR / week) |
|---|---|
| Mean weekly food budget | 195.2 |
| Min weekly food budget | 88.5 |
| Max weekly food budget | 330.9 |
| Mean optimized food cost | 24.4 |
| Mean budget utilization | 12.5% |

The optimizer consistently identifies meal plans that satisfy all nutritional constraints at a small fraction of the available food budget, demonstrating significant cost efficiency achievable through dietary optimization.

## 4. Price Shock Resilience (+20% Seafood Prices)

| Metric | Baseline | Post-Shock | Δ |
|---|---|---|---|
| Optimized cost (SAR/week) | 24.4 | 26.1 | +1.7 |
| Foods selected | 5 | 5 | 0 |
| Calories coverage (%) | 102.3 | 102.1 | −0.2 |
| Protein coverage (%) | 109.0 | 107.4 | −1.6 |
| Vitamin D coverage (%) | 108.2 | 106.9 | −1.3 |
| Iron coverage (%) | 103.4 | 103.4 | 0.0 |

The optimizer successfully adapts to price shocks by substituting equivalent nutritional alternatives, maintaining all constraints at marginal additional cost.

## Reproducing These Results

```bash
git clone https://github.com/aliakarma/FinNutriAgent.git
cd FinNutriAgent
pip install -r requirements.txt
jupyter notebook example/demo_finagent.ipynb
```

All figures and tables in the paper can be regenerated by running the notebook end-to-end.
